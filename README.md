# 💳 Credit Card Fraud Detection using Machine Learning

## 📌 Objective
The objective of this project is to detect fraudulent credit card transactions using Machine Learning techniques.

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## 📊 Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis
3. Fraud vs Normal Transaction Analysis
4. Logistic Regression Model Building
5. Model Evaluation
6. Data Visualization

## 📈 Outcome
Successfully built a fraud detection system capable of identifying fraudulent transactions with high accuracy.
